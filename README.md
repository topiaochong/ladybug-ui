# ladybug-ui 组件库

### 快速开始

#### 1.安装组件库

```bash
npm i ladybug-ui
```

### 2.引用组件库

```javascript
//全部引入
import "ladybug-ui/dist/css/index.css";
import LadyBugUI from "ladybug-ui";
Vue.use(LadyBugUI);

//按需引入
import "ladybug-ui/dist/css/demo.css";
import { Demo } from "ladybug-ui";
Vue.use(Demo);
```
