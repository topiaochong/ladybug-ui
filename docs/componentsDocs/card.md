# Card

卡片组件

### 示例

<m-card imgSrc="/logo.png" summary="测试"></m-card>

### 代码

```html
<m-card imgSrc="/logo.png" summary="测试"></m-card>
```

### Attributes

| 参数   | 说明         | 类型   | 是否必填 | 默认值 |
| ------ | ------------ | ------ | -------- | ------ |
| width  | 卡片的宽度   | Number | false    | -      |
| imgSrc | 图片资源地址 | String | true     | -      |
| footer | 卡片底部     | Slot   | false    | -      |
